DO converter 'respirometry'

This dissolved oxygen converter is graphical implementation of the function 'conv_O2' from the R package 'respirometry'. Light modifications to the function have been applied for being able to convert data from multichannel respirometry systems using 'AutoResp', 'FishResp' and 'Qbox-Aqua' oxygen logger formats.

Please, find more information on the conversion mechanism of DO units here:

Birk, M. (2018). respirometry: Tools for Conducting Respirometry Experiments, version 0.7.0, URL:
https://cran.r-project.org/package=respirometry
