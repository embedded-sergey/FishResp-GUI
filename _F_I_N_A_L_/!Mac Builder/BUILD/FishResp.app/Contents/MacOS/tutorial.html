Graphical implementation of R package 'FishResp' (version 1.0)

Author: Sergey Morozov

Java developers: Artur Zaripov & Roman Kovalev

Calculates metabolic rate of fish and other aquatic organisms measured using an intermittent-flow respirometry approach. The tool is used to run a set of graphical QC tests of raw respirometry data, correct it for background respiration and chamber effect, filter and extract target values of absolute and mass-specific metabolic rate. Experimental design should include background respiration tests and measuring of one or more metabolic rate traits. The package allows a user to import raw respirometry data obtained from 'AquaResp' (free software), 'AutoResp' ('LoligoSystems'), 'OxyView' ('PreSens'), 'Pyro Oxygen Logger' ('PyroScience') and 'Qbox-Aqua' ('QubitSystems').

Quick Guide

The software consists of five tabs with functional blocks on the left side and information or graphical modules on the right side of the application. The analysis of data goes step-by-step: from the left tab to the right one, top-down in each tab. Different colours of tabs and buttons are used as indicators of progress:

- the action is completed

- the action is required

- the action is unavailable

- plot a graph (no effect on data)


The manual for this software is under writing. For more information on the software, please visit the official website: fishresp.org

* DO saturation and partial pressure units can be converted to DO concentration units in graphical implementation of R packages 'rMR' and 'respirometry' (see Extensions module)
