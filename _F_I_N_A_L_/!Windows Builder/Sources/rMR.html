DO converter 'rMR'

This dissolved oxygen converter is graphical implementation of the function 'DO.unit.converter' from the R package 'rMR'. Light modifications to the function have been applied for being able to convert raw data from multichannel respirometry systems using 'AutoResp', 'FishResp' and 'Qbox-Aqua' oxygen logger formats.

Please, find more information on the conversion mechanism of DO units here:

Moulten, T. (2018). Importing Data from Loligo Systems Software, Calculating Metabolic Rates and Critical Tensions, version 1.1.0, URL: https://cran.r-project.org/package=rMR
