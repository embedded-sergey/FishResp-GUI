PreSens to FishResp

This extension is graphical implementation of the function 'presens.aquaresp' from the R package 'FishResp' used to convert raw data from 'OxyView' (PreSens) and a summary file from 'AquaResp' (free software) to 'FishResp' format.

Please, find more information on the package 'FishResp' and particularly the function 'presens.aquaresp' in the manual: https://cran.r-project.org/package=FishResp
