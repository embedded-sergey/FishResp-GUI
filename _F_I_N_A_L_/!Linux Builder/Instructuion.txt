Собрать установочный архив:
	tar -czvf FishResp.tar.gz FishResp/
Как устанавливать:
1) распаковать: 
	tar -xvf FishResp.tar.gz
2) установить в систему (по надобности - позволяет вызывать программу из консоли командой FishResp) с помощью скрипта install.sh, так же скрипт производит все проверки, настройки и скачивает необходимые библиотеки для R
2.1) вызвать команду для определения директории в терминале:
	cd <адрес директории с файлом install.sh>
2.2) вызвать команду для установления прав на запуск установщику:
	chmod u+x install.sh
2.3) запустить скрипт:
	./install.sh
2.4) ввести рут пароль, ибо пакет rJava для конфигурации на linux требует рут прав, попутно рут права используются чтоб 100% запихать программу по адресу /usr/lib (даже если у пользователя нет на эту директорию прав)
3) запустить программу:
	FishResp

- Опционально создать ярлык. Чуть позже запилю эту тему.
Первая папка FishResp - для создания архива
Вторая папка FishResp - для копирования в /usr/lib
